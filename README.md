# D43-RaivisBajars-AutoRezervesDaluInternetveikals
## Projekta apraksts
Šis ir PIKC "Rīgas Valsts tehnikums" kvalifikācijas darbs auto rezerves daļu internetveikals.

## Izmantotās tehnoloģijas
Projekta tiek izmantots:
  - Laravel
  - Vuejs
    - Vue Router
    - Vue Axios
    - BootstrapVue
  - Composer
  - Mysql
  - SCSS
  - CSS
